# 14_JVM (Java Virtual Machine)

> 자바 가상 머신

JVM은 JRE에 포함되어 Java 프로그램을 실행시키기 위한 환경 중 하나이다. 따라서 JRE가 설치되어 있으면 JVM도 설치되어 있다.

JVM은 이름 그대로, 자바를 실행시키기 위한 가상 머신이라고 생각하면 된다.
